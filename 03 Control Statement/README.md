# 📘 Control Statements

This module provides a comprehensive introduction to **Python Control Statements**, which are used to control the flow of execution in a program. Control statements enable programs to make decisions, repeat tasks, and alter the normal execution of code based on conditions.

The module is organized into three major categories: **Selection Statements**, **Iteration Statements**, and **Jump Statements**. Each category contains notebooks that explain the concepts in detail with syntax, examples, outputs, and practical use cases.

---

## ❓ Why Learn Control Statements?

Control statements are one of the most important concepts in programming. They help you:

- Make decisions based on conditions.
- Execute code repeatedly using loops.
- Control the flow of program execution.
- Write dynamic and efficient programs.
- Solve real-world programming problems.

---

## 📂 Module Structure

### 1. Selection Statements

Selection statements allow a program to execute different blocks of code depending on whether a condition is true or false.

**Topics Covered**

- If Statement
- If Else Statement
- If Elif Else Statement

---

### 2. Iteration Statements

Iteration statements allow a block of code to execute repeatedly until a condition is met or for every item in a sequence.

**Topics Covered**

- While Loop
- For Loop

---

### 3. Jump Statements

Jump statements alter the normal flow of loops or program execution.

**Topics Covered**

- Break Statement
- Continue Statement
- Pass Statement

---

## 📊 Control Statement Categories

| Category | Statements |
|----------|------------|
| Selection Statements | `if`, `if...else`, `if...elif...else` |
| Iteration Statements | `while`, `for` |
| Jump Statements | `break`, `continue`, `pass` |

For a better understanding of Python Control Statements, study the topics in the following order:
