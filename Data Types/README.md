# 📘 Data Types

This module provides a comprehensive introduction to **Python Data Types**. It is organized into multiple sections based on Python's built-in data type categories. Each section contains an **Introduction.ipynb** notebook that explains the concepts in detail, along with additional notebooks or folders dedicated to specific data types.

## 📂 Module Structure

### 1. Numeric Data Types

This section introduces Python's numeric data types and explains their characteristics, operations, and usage.

**Topics Covered**

- Integer (`int`)
- Floating Point (`float`)
- Complex (`complex`)
- Boolean (`bool`)

---

### 2. Sequence Data Types

This section explains sequence data types, which store ordered collections of values.

**Topics Covered**

- String (`str`)
- List (`list`)
- Tuple (`tuple`)
- Range (`range`)

---

### 3. Set Data Types

This section introduces unordered collections of unique elements.

**Topics Covered**

- Set (`set`)
- Frozen Set (`frozenset`)

---

### 4. Mapping Data Types

This section explains Python's mapping data type used for storing data in key-value pairs.

**Topics Covered**

- Dictionary (`dict`)

---

### 5. Binary Data Types

This section introduces Python's binary data types, which are primarily used for handling binary data and memory-efficient operations.

**Topics Covered**

- Bytes (`bytes`)
- Bytearray (`bytearray`)
- Memory View (`memoryview`)

---

### 6. Special Data Type

This section explains Python's special data type.

**Topics Covered**

- NoneType

---

## 📖 Recommended Learning Order

For a better understanding of Python Data Types, study the sections in the following order:

1. Numeric Data Types
2. Sequence Data Types
3. Set Data Types
4. Mapping Data Types
5. Binary Data Types
6. Special Data Type

Following this order provides a gradual understanding of Python's built-in data types, starting from basic numeric values to more advanced data structures.
